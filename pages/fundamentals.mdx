## Les fondamentaux

Dans cette section, vous allez voir les fondamentaux pour devenir un controlleur aériens virtuel.

Bien que ce guide soit majoritairement écrit en français, il est important de noter que la langue de communication principale dans le monde de la simulation aérienne est l'anglais.
Il est donc important de comprendre et de parler cette langue pour devenir un contrôleur aérien virtuel.

<Callout type="info">
  Vous trouverez pour chaque exemple, la correspondance en anglais afin de vous
  familiariser avec les termes utilisés.
</Callout>

Afin d'appronfondir vos connaissances, je vous conseille de consulter le [Skybrary](https://www.skybrary.aero), une encyclopédie en ligne sur la sécurité aérienne.
