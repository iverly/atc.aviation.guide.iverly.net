## Comment vous inscrire sur IVAO ?

Avant de vous inscrire sur IVAO, vous **devez** avoir lu les règles et procédures d'IVAO disponibles sur le site web d'IVAO. Vous pouvez les trouver [ici](https://wiki.ivao.aero/en/home/ivao/information).

<Callout type="error">
  Si vous ne respectez pas les règles et procédures d'IVAO, vous risquez d'être
  banni de leur réseau.
</Callout>

De plus, pour vous inscrire sur IVAO, vous devez respectez les conditions suivantes :

- Avoir au moins 14 ans
- Avoir une adresse email valide
- Utiliser votre vrai nom et prénom

Si vous respectez ces conditions, vous pouvez vous inscrire sur IVAO en vous rendez sur le lien suivant :

[https://ivao.aero/members/person/register.htm](https://ivao.aero/members/person/register.htm)

Une fois que vous avez rempli le formulaire d'inscription, vous recevrez un email de confirmation. Vous devrez cliquer sur le lien de confirmation pour activer votre compte.
