## Quels logiciels utiliser faire du contrôle sur IVAO ?

Il existe plusieurs logiciels développés par IVAO pour faire du contrôle aérien. Certains sont plus ancien que d'autres, mais ils sont tous fonctionnels. Voici la liste des logiciels :

- **Aurora** : C'est le dernier logiciel de contrôle aérien développé par IVAO. Il est disponible pour Windows, Mac et Linux. Vous pouvez le télécharger [ici](https://ivao.aero/softdev/software/aurora.asp).
- **IvAc** : C'est l'ancien logiciel de contrôle aérien d'IVAO. Il n'est plus disponible au téléchargement, mais il est toujours fonctionnel.

<Callout type="info">
  La suite de ce guide est exclusivement dédiée à l'utilisation d'Aurora. Si
  vous utilisez IvAc, vous pouvez vous référer à la documentation officielle
  d'IVAO.
</Callout>

Vous n'avez pas besoin d'installer un logiciel de discussion vocale pour contrôler sur IVAO. En effet, Aurora intègre un système de discussion vocale.

## Comment installer Aurora ?

Pour installer Aurora, vous devez télécharger le fichier d'installation depuis le site web d'IVAO. Voici les étapes à suivre :

1. Rendez-vous sur la page de téléchargement d'Aurora en cliquant [ici](https://ivao.aero/softdev/software/aurora.asp).
2. Cliquez sur le bouton "Download" pour télécharger le fichier d'installation pour votre OS.
3. Une fois le fichier téléchargé, ouvrez-le et suivez les instructions pour installer Aurora.

## Comment configurer Aurora ?

Une fois Aurora installé, vous devez le configurer pour qu'il fonctionne correctement.

Pour cette étape, je vous conseille de suivre le manuel d'utilisation d'Aurora. Vous pouvez le trouver [ici](https://wiki.ivao.aero/fr/home/devops/manuals/Aurora_Manual).
